## Описание решения
Текущее решение (solution) служит стендом для изучения различных технологий и подходов.


### Проекты *.Domain
Все эти проекты помогают тестировать гипотезы проектирования доменных моделей, описания их бизнес логики, применения value-object-ов и т.д.
Отдельно стоит отметить, что в рамках данных проектов также совершаются попытки выделить хранимое состояние доменной модели и использовать его совместно с EF Core и реляционной БД Postgres.
Ранее был опыт сохранения состояния модели только в NoSQL хранилище (MongoDB).

### Проект SpecFlowTests.Domain.Tests
Данный проект предназначен для изучения и демонстрации подхода к разработке с использованием исполняемой спецификации (Specflow). Подробнее можно почитать в README самого проекта.
